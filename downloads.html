Download Pyccuracy Source Code

Linux
Open up a terminal window, go to the folder you want the source code to be and type:
            git clone git://github.com/heynemann/pyccuracy.git

Windows
Open up a command window, go to the folder you want the source code to be and type:
            git clone git://github.com/heynemann/pyccuracy.git


Download Stable Source Code

Python Package Index
You can use Python Package Index (PyPI) to download Pyccuracy's latest stable release.

Just go to Pyccuracy's PyPI page and download the release.
