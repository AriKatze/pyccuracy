Test Language DefinitionsÂ¶

Below you can find the different groups of actions that Pyccuracy supports.

Under each action group you can see the syntax used for performing some action. If you need more detail about it, just click the desired action, or click the action group to know more about it.

  • Button Actions Grammar
    • I see “some” button
    • I do not see “some” button
    • I click “some” button
    • I see “some” button is enabled
    • I see “some” button is disabled
    • I mouseover “some” button
  • Checkbox Actions Grammar
    • I see “some” checkbox
    • I do not see “some” checkbox
    • I see the “some” checkbox is checked
    • I see the “some” checkbox is not checked
    • I check the “some” checkbox
    • I uncheck the “some” checkbox
  • Div Actions Grammar
    • I see “some” div
    • I do not see “some” div
    • I see “some” div contains “some text”
    • I see “some” div contains “some” markup
    • I see “some” div does not contain “some text”
    • I mouseover “some” div
  • Image Actions Grammar
    • I see “some” image
    • I click “some” image
    • I see “some” image has src of “some.gif”
    • I mouseover “some” image
  • Link Actions Grammar
    • I see “some” link
    • I do not see “some” link
    • I click “some” link
    • I see “some” link is enabled
    • I see “some” link is disabled
    • I see “some” link has “some” text
    • I see “some” link has “some” href
    • I mouseover “some” link
  • Page Actions Grammar
    • I go to “some url”
    • I wait for the page to load
    • I see “some” title
    • I wait for 0.5 seconds
  • Select Actions Grammar
    • I see “some” select
    • I do not see “some” select
    • I see “some” select has selected index of 1
    • I see “some” select has selected value of “1”
    • I see “some” select has selected text of “Something”
    • I see “some” select does not have selected index of 1
    • I see “some” select does not have selected value of “1”
    • I see “some” select does not have selected text of “something”
    • I see “some” select is enabled
    • I see “some” select is disabled
    • I select the option with index of 1 in “some” select
    • I select the option with value of “1” in “some” select
    • I select the option with text of “something” in “some” select
  • Textbox Actions Grammar
    • I see “some” textbox
    • I fill “some” textbox with “some text”
    • I clean “some” textbox
    • I see “some” textbox contains “some text”
    • I see “some” textbox does not contain “some text”
    • I see “some” textbox is enabled
    • I see “some” textbox is disabled
    • I see “some” textbox matches “some text”
    • I see “some” textbox does not match “some text”
    • I see “some” textbox is empty
    • I see “some” textbox is not empty
